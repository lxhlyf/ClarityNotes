# ClarityNotes
<img src="./art/logo.png" width="128" alt="logo">

ClarityNotes客户端,是我的一个练手的习作，其中采用了litepel以及一些基于Material Design设计风格的库，采用架构是MVC实现上比较简单。


## Statement
本程序是本人独立开发。


## Features
- 首页五大模块: 英语词根笔记 / 英语短语笔记 / 美句笔记 / 读书笔记 / 日记
